Project Overview

Solving Real-World Pharmacy Challenges

The Problem

Pharmacies face significant challenges in managing complex inventories, tracking expiring medications, handling daily point-of-sale transactions efficiently, and managing customer credit lines without errors.

Our Solution

7X Pharma Core is a comprehensive, scalable, and secure management system tailored for pharmacies. It streamlines inventory, automates sales tracking, and centralizes patient credit management.

Core Kernel

The app/core folder acts as the engine, bootstrapping the application, parsing URLs via Router, and instantiating controllers.

Public Directory

All HTTP requests are directed to public/index.php via .htaccess. This isolates application logic securely outside the document root.

System Strengths

High Security

Uses PDO prepared statements to prevent SQL Injection, and password hashing for secure authentication. Cross-Site Scripting (XSS) protections implemented.

Optimal Performance

Custom lightweight MVC framework means zero bloat. Pages load instantly, and database queries are optimized with proper indexing.

Clean Architecture

Strict separation of concerns. UI modifications don't break business logic, and database schemas can be updated independently.

Premium UI/UX

Designed with a modern, glassmorphism aesthetic using Tailwind CSS. Provides an intuitive and professional user experience.
