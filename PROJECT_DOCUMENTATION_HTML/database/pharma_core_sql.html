pharma_core.sql

ملف بناء قاعدة البيانات وتأسيس الهيكلة (Seeder File)

الرئيسية / Database / pharma_core.sql

محتويات الملف

هذا الملف هو المسؤول عن بناء قاعدة البيانات بالكامل. لا يقتصر فقط على إنشاء الجداول، بل يقوم أيضاً بحقن بيانات أولية (Seeding) لتجربة النظام مباشرة.

DDL Data Definition Language

يحتوي على أوامر CREATE TABLE لإنشاء الـ 7 جداول الرئيسية: المستخدمين (users)، الأدوية (medicines)، الدفعات (batches)، العملاء (clients)، المبيعات (sales)، تفاصيل المبيعات (sale_items)، ومدفوعات العملاء (client_payments).

DML Data Manipulation Language

يحتوي على أوامر INSERT INTO لإضافة مدير افتراضي للنظام، بالإضافة إلى إدخال أكثر من 30 صنفاً دوائياً حقيقياً مع أسعارها وتصنيفاتها وكمياتها لتسهيل عملية الاختبار والتقييم.

Constraints Foreign Keys

يحدد العلاقات بين الجداول. على سبيل المثال، يربط جدول المبيعات بالمستخدم الذي قام بالبيع، ويربط تفاصيل المبيعات بالدواء المباع. تم استخدام ON DELETE RESTRICT لبعض العلاقات لمنع حذف أي عنصر (كالدواء أو العميل) إذا كان مرتبطاً بفواتير سابقة، وذلك لضمان السلامة المالية للنظام.
