public getRecentSales($limit = 5)

استعلام معقد وشامل (Comprehensive Query) مصمم لجلب أحدث المبيعات للداشبورد.

  • LPAD: دالة MySQL تقوم بتحويل رقم المعرّف (1) إلى رقم فاتورة احترافي مثل RX-000001.
  • COALESCE مع LEFT JOIN: إذا كان العميل مسجلاً client_id يظهر اسمه، وإن كان غير مسجل NULL يطبع 'Walk-in Customer' (زبون عابر).
  • Subquery (استعلام فرعي): يحسب إجمالي عدد العلب/العناصر داخل الفاتورة مباشرة من جدول sale_items باستخدام SELECT SUM(quantity) ليظهر كـ items_count.
  • CASE WHEN: يولد حقل status بحيث إذا كانت طريقة الدفع هي الدين (credit) يجعله 'Pending'، وغير ذلك 'Paid'.
