Credit Controller

متحكم دفتر الديون وإدارة حسابات العملاء

الرئيسية / Controllers / CreditController.php

توضيح الدوال

public index() / add()

تجلب وتضيف العملاء من وإلى قاعدة البيانات (دفتر الكريدي).

public pay()

تسجل المبالغ التي يسددها العميل للصيدلية لتخفيض دينه (تسديد جزئي أو كلي) عبر استدعاء الموديل makePayment(). تستقبل القيمة، نوع الدفع، والملاحظات الإضافية وتمرر رقم الموظف المسؤول.

public history($id)

تفتح ملفاً شخصياً للعميل المحدد يعرض تاريخ الدفعات السابقة بالكامل وتحسب إجمالي المبالغ المسددة لتوضيح حركة الحساب بشكل شفاف.
