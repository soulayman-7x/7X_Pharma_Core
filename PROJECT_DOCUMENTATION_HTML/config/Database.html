🔍 شرح DSN (Data Source Name):

mysql:host=127.0.0.1;port=3306;dbname=pharma_core;charset=utf8mb4

  • mysql: — بروتوكول قاعدة البيانات
  • host=127.0.0.1 — عنوان الخادم (127.0.0.1 أفضل من localhost لتجنب تأخير DNS)
  • port=3306 — المنفذ الافتراضي لـ MySQL
  • charset=utf8mb4 — يدعم كل Unicode بما في ذلك الإيموجي

💡 شرح خيارات PDO الثلاثة:

  • ERRMODE_EXCEPTION — يُلقي استثناء (Exception) عند أي خطأ SQL بدلاً من الصمت
  • FETCH_ASSOC — يُرجع البيانات كمصفوفة مفاتيح نصية فقط (بدون أرقام مكررة)
  • EMULATE_PREPARES = false — يستخدم Prepared Statements الحقيقية بدلاً من المحاكاة، أكثر أماناً
