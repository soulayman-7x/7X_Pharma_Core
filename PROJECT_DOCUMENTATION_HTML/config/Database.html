Database.php

فئة إدارة الاتصال بقواعد البيانات وتتبع الأخطاء

الرئيسية / Config / Database.php

وصف الملف

ملف Database.php هو المسؤول الحصري عن إنشاء وإدارة الاتصال بخادم MySQL باستخدام تقنية PDO. تم تصميمه بحيث يقرأ متغيرات البيئة $_ENV إن وجدت، ويوفر آلية قوية للتعامل مع أخطاء الاتصال (Error Handling) وتسجيلها في ملفات (Logs) دون إظهارها للمستخدم النهائي لحماية أمان السيرفر.

المتغيرات والدوال (Methods)

private $conn

متغير يحمل كائن الاتصال بـ PDO. لا يمكن الوصول إليه من خارج الكلاس للحفاظ على التغليف (Encapsulation).

public __construct()

طريقة العمل:

  1. يتم استخراج الإعدادات (المضيف، المنفذ، اسم قاعدة البيانات، المستخدم، كلمة المرور) من $_ENV أو استخدام القيم الافتراضية.
  2. يتم بناء سلسلة الاتصال (Data Source Name - DSN) وتحديد الترميز charset=utf8mb4 لدعم جميع اللغات والرموز التعبيرية.
  3. ضبط الخيارات (Options):
    • ATTR_ERRMODE => ERRMODE_EXCEPTION: إجبار PDO على إطلاق استثناءات (Exceptions) عند حدوث أي خطأ برمجي.
    • ATTR_DEFAULT_FETCH_MODE => FETCH_ASSOC: ليكون ناتج الاستعلامات مصفوفة ترابطية (مفتاح وقيمة).
    • ATTR_EMULATE_PREPARES => false: إيقاف المحاكاة ودفع MySQL للقيام بعملية Prepare حقيقية لزيادة الأمان والأداء.
  4. يتم محاولة الاتصال عبر كتلة try...catch.

public getConnection()

الوظيفة: تُرجع كائن الاتصال المخزن في $this->conn ليستفيد منه كلاس Model.php وباقي النماذج.

نظام تتبع ومعالجة الأخطاء (Error Handling)

private logError($e)

عند فشل الاتصال بقاعدة البيانات، يقوم كاتش catch(PDOException $e) بالتقاط الخطأ وتنفيذ الآتي عبر الدالة logError:

  • التحقق من وجود مجلد logs/ في مسار النظام الأصلي (ROOT_DIR).
  • إذا لم يكن موجوداً، يقوم النظام بإنشائه أوتوماتيكياً بصلاحيات 0755.
  • تسجيل رسالة الخطأ التقنية، وقت الحدوث، واسم الملف ورقم السطر في ملف db.errors.log بشكل تتابعي (APPEND).

حماية المستخدم (User Fallback)

بدلاً من عرض خطأ Fatal Error القبيح والذي قد يكشف مسارات السيرفر وكلمات المرور للمتسللين، يقوم الكود باستدعاء واجهة مخصصة للأخطاء views/components/db_error.php لعرض رسالة ودية للمستخدم، وإرسال حالة (HTTP 500 Internal Server Error) لتعرفها محركات البحث والمتصفحات. وإذا فُقد هذا الملف، يقوم بطباعة رسالة نصية بسيطة ثم إيقاف النظام بـ exit().
