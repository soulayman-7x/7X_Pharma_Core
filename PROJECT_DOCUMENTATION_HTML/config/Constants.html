Constants.php

الثوابت العامة وإعدادات المسارات في النظام

الرئيسية / Config / Constants.php

وصف الملف

ملف Constants.php هو أول ملف يتم استدعاؤه في الـ index.php. يقوم بتعريف الثوابت (Constants) العامة التي تُستخدم في مختلف أنحاء النظام. يُسهل هذا الملف عملية نقل المشروع من بيئة التطوير (Localhost) إلى بيئة الإنتاج (Production server) بتعديل المتغيرات في مكان واحد فقط.

define BASE_URL

يحدد الرابط الأساسي للموقع (مثال: http://localhost/7X_Pharma_Core/public). مفيد جداً عند استدعاء ملفات الـ CSS أو الـ JS أو عند إعادة التوجيه (Redirects). يأخذ القيمة إما من ملف الـ .env أو يستخدم القيمة الافتراضية.

define ROOT_DIR

يحدد المسار المادي الجذري (Absolute Path) لمجلد المشروع في نظام التشغيل (مثلاً C:\xampp\htdocs\7X_Pharma_Core). يتم استخدام دالة dirname(__DIR__, 2) للرجوع مجلدين إلى الخلف. وهذا يمنع مشاكل المسارات النسبية (Relative paths) عند استدعاء ملفات الـ PHP باستخدام require_once.

define APP_NAME

اسم التطبيق "7X Pharma Core". يُستخدم في ترويسات الصفحات (Titles) والواجهات لتسهيل تغييره مستقبلاً إذا دعت الحاجة.

define ROLE_ADMIN / ROLE_PHARMACIST

ثوابت تحدد مسميات الصلاحيات (Roles) الموجودة في النظام لتفادي الأخطاء الإملائية (Typo Errors) عند كتابتها في الجمل الشرطية للـ Authorization.
